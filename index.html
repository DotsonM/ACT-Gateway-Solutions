Errors and Solutions

System Error Resolution Case Number Examples
All Systems 2-In-1 (Tablet Mode)
  • Follow the instructions in Enable and disable a touchscreen in Windows - Microsoft Support to disable the touchscreen.
  • Locate the ACT Gateway shortcut or executable:
    • Navigate to the desktop or folder where the ACT Gateway application is located.
  • Right-click the ACT Gateway icon and select Run as administrator from the context menu.
  • When the User Account Control (UAC) prompt appears, it will ask: Do you want to allow this app to make changes to your device?
  • Enter admin credentials:
    • Provide a username and password belonging to a user with administrative privileges.
  • Click Yes or OK to proceed.
  • The ACT Gateway application will launch with elevated privileges, allowing access to protected system resources or configurations.
All Systems Error initializing the test

Steps to Resolve Student Room Errors

  • Make sure the student room is not paused; if paused, unpause it.
  • Click the Troubleshoot button next to the student's name and wait a moment.
  • If the error persists:
    • Have the student and test coordinator reboot their devices.
    • Try using a different device if one is available.
  • If the error still occurs:
    • Have the Technical Coordinator uninstall the Gateway application and reinstall it on the student's device.
All Systems Failed to fetch data This error should not be coming up anymore for test coordinators. If it does, we will let our immediate supervisor know and reach out to the internal technical team.
All Systems JavaScript errors after logging in
  • Clear App cache and check application version
  • Uninstall Gateway
  • Reboot and reinstall

    • Group 1: Clear System Cache

    • To clear your Windows cache:
      • Use Disk Cleanup for system files.
      • Use the Settings app for temporary files.
      • Use your web browser's settings for browsing data.
    • For DNS cache:
      • Open Command Prompt as an administrator.
      • Type: ipconfig /flushdns and press Enter.

    Web Browsers

    • Open your browser (Chrome, Edge, Firefox, etc.).
    • Access the browser's history or settings menu.
    • Select the option to clear browsing data.
    • Choose "Cached images and files" (or similar) and set the time range to "All time".
    • Click Clear to remove the cache.

    For Temporary and System Files

    • Open the Start Menu and type "Disk Cleanup".
    • Select Clean up system files to see more options.
    • Check the boxes next to file types like:
      • Temporary Internet Files
      • Temporary files
      • DirectX cache
    • Click OK to delete them.

    For Specific App Cache (e.g., Microsoft Teams)

    • Search for and open the Settings app.
    • Go to Apps > Installed apps.
    • Find the app (e.g., Microsoft Teams), select the three-dots menu, and click Advanced options.
    • In the Reset section, select Reset and restart the app.

    For DNS Cache

    • Open the Windows search bar and type "cmd".
    • Right-click on Command Prompt and select Run as administrator.
    • In the Command Prompt window, type:
      • ipconfig /flushdns
    • Press Enter.
