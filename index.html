Errors and Solutions

System Error Resolution Case Number Examples
All Systems 2-In-1 (Tablet Mode)
  • Follow the instructions in Enable and disable a touchscreen in Windows - Microsoft Support to disable the touchscreen.
  • Locate the ACT Gateway shortcut or executable:
    • Navigate to the desktop or folder where the ACT Gateway application is located.
  • Right-click the ACT Gateway icon and select Run as administrator from the context menu.
  • When the User Account Control (UAC) prompt appears, it will ask: Do you want to allow this app to make changes to your device?
  • Enter admin credentials:
    • Provide a username and password belonging to a user with administrative privileges.
  • Click Yes or OK to proceed.
  • The ACT Gateway application will launch with elevated privileges, allowing access to protected system resources or configurations.
All Systems Error initializing the test

Steps to Resolve Student Room Errors

  • Make sure the student room is not paused; if paused, unpause it.
  • Click the Troubleshoot button next to the student's name and wait a moment.
  • If the error persists:
    • Have the student and test coordinator reboot their devices.
    • Try using a different device if one is available.
  • If the error still occurs:
    • Have the Technical Coordinator uninstall the Gateway application and reinstall it on the student's device.
